What works

  • charging
  • network through USB
  • telnet in initramfs
  • ssh
  • adjusting the led, torch and the lcd backlight

What does not work

  • framebuffer
  • Wifi

What have not been tested

  • Bluetooth
  • 3G
  • Sound
  • Phone calls
  • SD card
  • Hardware buttons

Kernels

sonyxperiadev kernel

It's the most stable one, works everything listed above.

LineageOS kernel

Works everything listed above, has heating issues.

Mainline

Compilation and flashing is successfull, haven't tried UART.

Issues

framebuffer

Weston does not show up by default.

Workaround:

for i in 1 2; do echo "0,0" > /sys/class/graphics/fb0/pan; done

Run it when weston is already started.
